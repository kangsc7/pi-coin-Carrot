PG사 연동 가이드

1. 토스페이먼츠 연동:

  • 토스페이먼츠 개발자센터에서 계정 생성
  • 클라이언트 키와 시크릿 키 발급
  • 테스트 환경에서 충분히 테스트 후 운영 키로 전환

2. 카카오페이 연동:

  • 카카오 개발자센터에서 앱 등록
  • 카카오페이 서비스 활성화
  • 가맹점 코드(CID) 및 시크릿 키 확인

주의사항: 실제 운영 환경에서는 반드시 HTTPS를 사용하고, 시크릿 키는 안전하게 관리해주세요.
